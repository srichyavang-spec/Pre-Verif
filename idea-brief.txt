Build an AI-powered Government Application Assistant MCP server that helps citizens complete government service applications efficiently and securely. The server should integrate with supported government services (such as DigiLocker where available) and user-uploaded documents to verify certificates, identify missing or invalid documents, perform multi-step verification using official government data sources, and maintain a structured document repository. It should automatically populate supported government application forms, validate entered information, convert uploaded documents into required formats, provide detailed step-by-step guidance for each application, fetch the latest eligibility rules and notifications from official government websites, monitor application status, and notify users of updates. Implement the solution using MCP Tools for verification, document processing, form filling, and status tracking; MCP Resources for storing application data, document metadata, and government requirements; and MCP Prompts to guide users through each service. Ensure secure authentication, explicit user consent, privacy protection, audit logging, and human confirmation before any external submission or sensitive action. Design the project as a production-ready, modular MCP server with a clean architecture, scalable components, and a modern user interface suitable for real-world deployment.
